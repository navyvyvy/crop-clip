# CropClip 개인정보처리방침

CropClip은 사용자가 치지직 방송 화면에서 직접 선택한 영역만 로컬로 녹화하고, 결과를 미리보기/저장/분할/변환할 수 있도록 돕는 Chrome 확장 프로그램입니다.

## 수집 및 저장하는 데이터

CropClip은 기능 제공을 위해 사용자의 브라우저 안에 다음 데이터를 저장할 수 있습니다.

- 녹화 설정: 저장 형식, 비트레이트, 60fps 설정
- 선택 영역 정보: 사용자가 지정한 녹화 영역의 위치와 크기
- 녹화 상태: 녹화 진행 여부와 결과 화면 연결에 필요한 임시 상태
- 녹화 결과 파일: 사용자가 직접 녹화한 파일과 분할된 파일

## 사용 목적

저장된 데이터는 다음 목적에만 사용됩니다.

- 사용자가 선택한 화면 영역을 녹화하기 위해
- 녹화 결과를 미리보기 및 저장할 수 있게 하기 위해
- 녹화 파일을 시간 또는 용량 기준으로 나누기 위해
- 녹화 결과를 MP4 또는 GIF 형식으로 변환하기 위해
- 결과 창이 닫힌 뒤 임시 녹화 데이터를 자동 삭제하기 위해

## 외부 전송

CropClip은 사용자의 녹화 파일, 웹사이트 콘텐츠, 설정, 선택 영역 정보를 외부 서버로 전송하지 않습니다.

모든 녹화, 미리보기, 파일 나누기, 변환 처리는 사용자의 브라우저 안에서만 수행됩니다.

## 제3자 제공

CropClip은 사용자 데이터를 판매하거나 제3자에게 제공하지 않습니다.

## 보관 및 삭제

녹화 결과 데이터는 사용자의 브라우저 로컬 저장소에 임시로 저장됩니다. 결과 창이 닫히면 일정 시간이 지난 뒤 자동 삭제가 예약됩니다.

사용자는 브라우저의 확장 프로그램 데이터 삭제 또는 확장 프로그램 제거를 통해 저장된 데이터를 삭제할 수 있습니다.

## 원격 코드

CropClip은 외부 서버에서 실행 코드를 다운로드하거나 원격 스크립트를 실행하지 않습니다. JavaScript, CSS, WebAssembly 및 아이콘 리소스는 확장 프로그램 패키지 내부에 포함되어 있습니다.
