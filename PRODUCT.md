# Product

## Register

product

## Users

방송이나 영상에서 본인이 필요한 장면을 빠르게 녹화하고 저장하려는 사용자. 복잡한 편집 프로그램보다 브라우저 안에서 영역 선택, 확인, 저장을 짧게 끝내는 흐름을 선호한다.

## Product Purpose

사용자가 영상 화면의 필요한 영역과 시간을 직접 선택하고, 결과를 즉시 미리 본 뒤 원하는 형식으로 저장할 수 있게 한다. 성공 기준은 녹화부터 결과 확인과 다운로드까지 별도 편집 도구 없이 끝나는 것이다.

## Brand Personality

직관적, 단정함, 신뢰감. 기능 상태와 처리 결과를 숨기지 않고 짧고 정확한 한국어로 안내한다.

## Anti-references

전문 편집기처럼 패널과 용어가 과도한 화면, 장식이 작업보다 앞서는 화면, 현재 적용 범위를 알기 어려운 조작 방식, 버튼마다 색과 형태가 달라지는 인터페이스를 피한다.

## Design Principles

- 선택한 영역과 시간이 곧 결과물이라는 규칙을 모든 작업에 일관되게 적용한다.
- 미리보기에서 현재 상태를 먼저 보여주고 필요한 조작을 바로 아래에 둔다.
- 다운로드, 변환, 분할은 같은 파일과 같은 범위를 사용한다.
- 처리 중 상태와 실패 이유를 사용자가 바로 확인할 수 있게 한다.
- 한 화면에서 핵심 작업을 끝내되 작은 화면에서는 자연스럽게 세로로 재배치한다.

## Accessibility & Inclusion

키보드 조작과 명확한 포커스 표시를 제공하고, 색상만으로 선택 상태를 전달하지 않는다. 텍스트와 컨트롤은 다크 배경에서 충분한 대비를 유지하며 모션 감소 설정을 존중한다.
