# CropClip

[![Chrome Web Store에서 설치](https://developer.chrome.com/static/docs/webstore/branding/image/tbyBjqi7Zu733AAKA5n4.png)](https://chromewebstore.google.com/detail/idjkjkfehhhaeacngchpolnajodfibil)

CropClip은 영상이나 방송 화면에서 원하는 영역만 선택해 녹화하는 화면 크롭 녹화 확장 프로그램입니다. 

치지직과 유튜브에서 빠르게 영역을 지정할 수 있고, 녹화한 파일은 바로 미리보기와 다운로드로 확인할 수 있습니다.

복잡한 편집 없이 필요한 화면만 간단하게 잘라 녹화하고 싶은 사용자에게 맞춘 도구입니다.

## 주요 기능

- 치지직과 유튜브 플레이어 오른쪽 컨트롤 영역에 `클립 영역 선택` 아이콘이 표시됩니다.
- 영상 안에서 녹화할 영역을 직접 선택할 수 있습니다.
- 선택한 영역만 잘라 WebM 또는 브라우저가 지원하는 MP4 형식으로 녹화합니다.
- 선택한 영역의 현재 화면을 스크린샷으로 캡처하고 PNG로 저장할 수 있습니다.
- 녹화 결과 화면에서 영상을 바로 미리 볼 수 있습니다.
- 원본 녹화 파일을 다운로드할 수 있습니다.
- 결과 파일을 MP4 또는 GIF로 변환해 다운로드할 수 있습니다.
- 결과 파일을 시간 기준 또는 예상 용량 기준으로 나눌 수 있습니다.
- 나눈 파일은 개별 다운로드와 모두 다운로드를 지원합니다.
- 결과 창을 닫으면 녹화 파일은 일정 시간 뒤 자동 삭제되도록 예약됩니다.

## 기본 동작

- 기본 저장 형식은 WebM입니다.
- 기본 비트레이트는 4 Mbps입니다.
- 기본 프레임은 30fps이며, 옵션에서 60fps를 켤 수 있습니다.
- 영상이 음소거 상태이면 녹화를 시작하지 않습니다.
- 치지직과 YouTube는 기본 허용 사이트로 등록되어 있습니다.

## 패치노트

### v0.1.1

- 치지직에서 검색이나 화면 전환으로 플레이어 크기가 바뀌어도 선택한 영역 비율을 유지하고 녹화가 잘못 종료되지 않도록 개선했습니다.
- 다른 채널 이동, 방송 종료, 영상 재생 종료처럼 녹화 중인 영상 스트림이 끝나는 경우 자동으로 녹화를 마치도록 개선했습니다.
- 선택 영역 스크린샷 캡처와 PNG 저장 기능을 추가했습니다.
