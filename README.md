# CropClip

[![Chrome Web Store에서 설치](https://developer.chrome.com/static/docs/webstore/branding/image/tbyBjqi7Zu733AAKA5n4.png)](https://chromewebstore.google.com/detail/idjkjkfehhhaeacngchpolnajodfibil)

CropClip은 방송 화면에서 원하는 영역만 선택해 녹화하는 화면 녹화 확장 프로그램입니다.

치지직에서 빠르게 영역을 지정할 수 있고, 녹화한 파일은 바로 미리보기와 저장으로 확인할 수 있습니다.

복잡한 편집 없이 필요한 화면만 간단하게 잘라 녹화하고 싶은 사용자에게 맞춘 도구입니다.

## 주요 기능

- 치지직 플레이어 오른쪽 컨트롤 영역에 `녹화 영역 선택` 아이콘이 표시됩니다.
- 방송 화면 안에서 녹화할 영역을 직접 선택할 수 있습니다
- 선택한 영역만 잘라 WebM 또는 브라우저가 지원하는 MP4 형식으로 녹화합니다.
- 녹화 결과 화면에서 결과 파일을 바로 미리 보고 저장할 수 있습니다.
- 결과 파일을 시간 기준 또는 예상 용량 기준으로 나누거나 MP4 또는 GIF로 변환해 저장할 수 있습니다.
- 선택 영역 스크린샷 캡처와 PNG 저장 기능을 지원합니다.

## 기본 동작

- 기본 저장 형식은 WebM입니다.
- 기본 비트레이트는 4 Mbps입니다.
- 기본 프레임은 30fps이며, 옵션에서 60fps를 켤 수 있습니다.
- 방송이 음소거 상태이면 녹화를 시작하지 않습니다.
- 치지직은 기본 허용 사이트로 등록되어 있습니다.

## 패치노트

### v0.1.2

- 치지직 플레이어에서 전체 화면 녹화와 전체 스크린샷을 바로 실행할 수 있는 버튼을 추가했습니다.
- 팝업에서 전체 녹화 버튼과 전체 스크린샷 버튼 표시 여부를 각각 설정할 수 있도록 개선했습니다.

### v0.1.1

- 치지직에서 검색이나 화면 전환으로 플레이어 크기가 바뀌어도 선택한 영역 비율을 유지하고 녹화가 잘못 종료되지 않도록 개선했습니다.
- 다른 채널 이동, 방송 종료처럼 녹화 중인 스트림이 끝나는 경우 자동으로 녹화를 마치도록 개선했습니다.
- 선택 영역 스크린샷 캡처와 PNG 저장 기능을 추가했습니다.
