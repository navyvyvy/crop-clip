# CropClip

[![Chrome Web Store에서 설치](https://developer.chrome.com/static/docs/webstore/branding/image/tbyBjqi7Zu733AAKA5n4.png)](https://chromewebstore.google.com/detail/idjkjkfehhhaeacngchpolnajodfibil)

CropClip은 치지직 방송 화면에서 필요한 부분만 선택해 녹화하고, 결과를 바로 확인하고 저장할 수 있는 Chrome 확장 프로그램입니다.

녹화와 변환은 브라우저 안에서 처리되며 파일을 외부 서버로 전송하지 않습니다.

## 주요 기능

### 녹화와 캡처

- 치지직 플레이어의 `녹화 영역 선택` 버튼으로 방송 화면 안에서 원하는 영역을 지정할 수 있습니다.
- 선택한 영역 또는 플레이어 전체 화면을 WebM이나 브라우저가 지원하는 MP4 형식으로 녹화할 수 있습니다.
- 선택 영역과 플레이어 전체 화면을 PNG 스크린샷으로 저장할 수 있습니다.
- 다중영역을 켜면 최대 4개 영역을 선택하고 원본 위치를 반영해 하나의 영상으로 합칠 수 있습니다.
- 방송이 끝나거나 다른 채널로 이동하면 진행 중인 녹화를 자동으로 마칩니다.

### 결과 편집

- 녹화가 끝나면 결과 화면에서 영상을 바로 미리 보고 다운로드할 수 있습니다.
- 사용할 구간을 0.1초 단위로 선택하면 다운로드, 변환, 파일 나누기에 같은 범위가 적용됩니다.
- 파일을 시간 또는 예상 용량 기준으로 나누고 개별 파일이나 전체 파일을 순서대로 다운로드할 수 있습니다.
- 결과를 MP4 또는 GIF로 변환할 수 있습니다.
- 원본 영상은 그대로 보관되므로 언제든 전체 구간으로 되돌릴 수 있습니다.

### 조작 편의

- 영역을 이동하거나 크기를 조절할 때 다른 영역과 같은 선에 맞춰지는 가이드를 제공합니다.
- 좌우 방향키로 영상을 탐색할 수 있으며 이동 시간을 직접 설정할 수 있습니다.
- 자주 사용하는 기능은 한 글자 단축키로 실행하고 설정 화면에서 키를 변경할 수 있습니다.
- 결과창을 닫으면 녹화를 시작한 탭으로 돌아갑니다.

## 사용 순서

1. 치지직 플레이어에서 `녹화 영역 선택`을 누르고 영역을 지정합니다.
2. 선택 영역 위의 녹화 버튼 또는 확장 프로그램 창에서 녹화를 시작합니다.
3. 녹화를 정지하면 결과 화면에서 구간을 조절하고 다운로드합니다.

## 기본 단축키

| 키 | 기능 |
| --- | --- |
| `a` | 영역 선택 |
| `x` | 활성 영역 해제 |
| `z` | 전체 영역 해제 |
| `r` | 영역 녹화 시작·정지 |
| `c` | 녹화 취소 |
| `s` | 영역 스크린샷 |
| `e` | 전체 녹화 시작·정지 |
| `d` | 전체 스크린샷 |

단축키는 기능이 켜져 있을 때만 동작하며, 입력란에 글자를 입력하는 동안에는 실행되지 않습니다.

## 기본 동작

- 기본 저장 형식은 WebM입니다.
- 기본 비트레이트는 4 Mbps입니다.
- 기본 프레임은 30fps이며, 옵션에서 60fps를 켤 수 있습니다.
- 방송이 음소거 상태이면 녹화를 시작하지 않습니다.
- 다중영역, 전체 녹화, 전체 스크린샷, 단축키는 필요할 때 설정에서 켤 수 있습니다.
- 현재 치지직 사이트에서 사용할 수 있습니다.

## 패치노트

### v0.2.3

- 결과 화면에서 사용할 구간을 0.1초 단위로 조절하고 다운로드, 변환, 파일 나누기에 동일하게 적용하도록 추가했습니다.
- 선택 구간과 원본 파일을 구분해 다운로드할 수 있도록 개선했습니다.
- 시간·용량 프리셋, 변환 진행 상태, 초기 로딩 화면을 추가했습니다.
- WebM 미리보기 탐색과 구간 처리 성능, 변환 실패 처리를 안정화했습니다.

### v0.2.2

- 녹화 취소·완료·오류가 동시에 처리될 때 상태가 잘못 남는 문제를 개선했습니다.
- 방송 해상도가 바뀌어도 선택 영역의 합성 비율과 출력 크기를 유지하도록 개선했습니다.
- 다중영역의 가능한 가로·세로 분할을 재귀적으로 비교해 2~4개 영역의 혼합 배치를 원본 위치에 맞게 합성하도록 개선했습니다.
- 배율 변환 시 영역 경계를 같은 기준으로 계산해 합성 영상이 겹칠 수 있는 문제를 수정했습니다.
- 결과 파일 로딩과 변환 실패 처리, 임시 녹화 데이터 정리를 안정화했습니다.
- WebM 결과를 재생하지 않아도 원하는 시점으로 바로 이동할 수 있도록 개선했습니다.
- 확장 프로그램을 다시 로드해도 플레이어의 전체 녹화·스크린샷 버튼 설정이 바로 반영되도록 수정했습니다.

### v0.2.1

- 녹화 중 결과 파일을 만들지 않고 중단하는 녹화 취소 기능을 추가했습니다.
- 녹화 취소 단축키를 추가했습니다. 기본값은 `c`입니다.
- 다중영역 배치가 불필요하게 일렬로 붙지 않도록 영역 위치 판단을 개선했습니다.
- 영역 내부를 클릭하면 해당 영역이 바로 활성화되도록 개선했습니다.
- 전체 영역 해제 단축키를 추가했습니다. 기본값은 `z`입니다.
- 영역 선택 중 우클릭으로 선택을 취소할 수 있도록 개선했습니다.

### v0.2.0

- 다중영역 옵션을 추가해 최대 4개의 선택 영역을 함께 녹화할 수 있도록 했습니다.
- 팝업에서 다중영역 개수를 2~4개로 조절할 수 있도록 추가했습니다.
- 다중영역에서 `1`~`4` 키로 활성 영역을 바꾸고, 활성 영역은 더 진하게 표시되도록 개선했습니다.
- 영역 크기 조절 중 다른 영역의 같은 행/열 선과 맞으면 양쪽 선을 가이드 색으로 표시하고 스냅되도록 추가했습니다.
- 영역끼리 서로 겹치지 않도록 제한했습니다.
- 3개 이상 영역 녹화 시 위치 기준으로 영역을 묶어 더 자연스럽게 이어 붙이도록 개선했습니다.
