# CropClip

[![Chrome Web Store에서 설치](https://developer.chrome.com/static/docs/webstore/branding/image/tbyBjqi7Zu733AAKA5n4.png)](https://chromewebstore.google.com/detail/idjkjkfehhhaeacngchpolnajodfibil)

CropClip은 방송 화면에서 원하는 영역만 선택해 녹화하는 화면 녹화 확장 프로그램입니다.

치지직에서 빠르게 영역을 지정할 수 있고, 녹화한 파일은 바로 미리보기와 저장으로 확인할 수 있습니다.

복잡한 편집 없이 필요한 화면만 간단하게 잘라 녹화하고 싶은 사용자에게 맞춘 도구입니다.

## 주요 기능

- 치지직 플레이어 오른쪽 컨트롤 영역에 `녹화 영역 선택` 아이콘이 표시됩니다.
- 방송 화면 안에서 녹화할 영역을 직접 선택할 수 있습니다
- 선택한 영역만 잘라 WebM 또는 브라우저가 지원하는 MP4 형식으로 녹화합니다.
- 녹화 결과 화면에서 결과 파일을 바로 미리 보고 저장할 수 있습니다.
- 결과 화면에서 사용할 시간 구간을 직접 선택하면 미리보기, 다운로드, 변환, 파일 나누기에 같은 범위가 적용됩니다.
- 결과 파일을 시간 기준 또는 예상 용량 기준으로 나누거나 MP4 또는 GIF로 변환해 저장할 수 있습니다.
- 선택 영역 스크린샷 캡처와 PNG 저장 기능을 지원합니다.
- 좌/우 방향키로 이동할 수 있고 이동 초를 설정할 수 있습니다.
- 다중영역 옵션을 켜면 최대 4개의 영역을 선택해 함께 녹화할 수 있습니다.

## 기본 동작

- 기본 저장 형식은 WebM입니다.
- 기본 비트레이트는 4 Mbps입니다.
- 기본 프레임은 30fps이며, 옵션에서 60fps를 켤 수 있습니다.
- 방송이 음소거 상태이면 녹화를 시작하지 않습니다.
- 치지직은 기본 허용 사이트로 등록되어 있습니다.

## 패치노트

### v0.2.3

- 결과 화면에서 사용할 구간을 0.1초 단위로 조절하고 미리보기, 다운로드, 변환, 파일 나누기에 동일하게 적용하도록 추가했습니다.
- 구간 선택 시 원본과 선택 구간을 각각 다운로드할 수 있도록 개선했습니다.
- 파일 나누기에 시간·용량 프리셋과 진행 상태를 추가했습니다.
- 결과 파일을 준비하는 동안 빈 화면 대신 로딩 상태를 표시하도록 개선했습니다.
- WebM 미리보기 탐색과 변환 실패 처리를 안정화했습니다.

### v0.2.2

- 녹화 취소·완료·오류가 동시에 처리될 때 상태가 잘못 남는 문제를 개선했습니다.
- 방송 해상도가 바뀌어도 선택 영역의 합성 비율과 출력 크기를 유지하도록 개선했습니다.
- 다중영역의 가능한 가로·세로 분할을 재귀적으로 비교해 2~4개 영역의 혼합 배치를 원본 위치에 맞게 합성하도록 개선했습니다.
- 배율 변환 시 영역 경계를 같은 기준으로 계산해 합성 영상이 겹칠 수 있는 문제를 수정했습니다.
- 결과 파일 로딩과 변환 실패 처리, 임시 녹화 데이터 정리를 안정화했습니다.
- WebM 결과를 재생하지 않아도 원하는 시점으로 바로 이동할 수 있도록 개선했습니다.
- 확장 프로그램을 다시 로드해도 플레이어의 전체 녹화·스크린샷 버튼 설정이 바로 반영되도록 수정했습니다.

### v0.2.1

- 녹화 중 결과 파일을 만들지 않고 중단하는 녹화 취소 기능을 추가했습니다.
- 녹화 취소 단축키를 추가했습니다. 기본값은 `c`입니다.
- 다중영역 배치가 불필요하게 일렬로 붙지 않도록 영역 위치 판단을 개선했습니다.
- 영역 내부를 클릭하면 해당 영역이 바로 활성화되도록 개선했습니다.
- 전체 영역 해제 단축키를 추가했습니다. 기본값은 `z`입니다.
- 영역 선택 중 우클릭으로 선택을 취소할 수 있도록 개선했습니다.

### v0.2.0

- 다중영역 옵션을 추가해 최대 4개의 선택 영역을 함께 녹화할 수 있도록 했습니다.
- 팝업에서 다중영역 개수를 2~4개로 조절할 수 있도록 추가했습니다.
- 다중영역에서 `1`~`4` 키로 활성 영역을 바꾸고, 활성 영역은 더 진하게 표시되도록 개선했습니다.
- 영역 크기 조절 중 다른 영역의 같은 행/열 선과 맞으면 양쪽 선을 가이드 색으로 표시하고 스냅되도록 추가했습니다.
- 영역끼리 서로 겹치지 않도록 제한했습니다.
- 3개 이상 영역 녹화 시 위치 기준으로 영역을 묶어 더 자연스럽게 이어 붙이도록 개선했습니다.

### v0.1.3

- 파일명 앞에 스트리머 이름을 붙일 수 있는 옵션을 추가했습니다.
- 페이지 단축키로 영역 선택(a), 영역 해제(x), 영역 녹화(r), 영역 스크린샷(s), 전체 녹화(e), 전체 스크린샷(d)을 실행할 수 있도록 추가했습니다.
- 확장 프로그램 설정 창에서 단축키 변경, 단축키 초기화, 설정 전체 초기화를 실행할 수 있도록 추가했습니다.
- 좌/우 방향키 이동과 이동 초 설정을 추가했습니다.
- 녹화 결과창을 닫으면 이전 탭으로 돌아가도록 개선했습니다.
- 분할 파일 다운로드 시 순서대로 다운받도록 개선했습니다.

### v0.1.2

- 치지직 플레이어에서 전체 화면 녹화와 전체 스크린샷을 바로 실행할 수 있는 버튼을 추가했습니다.
- 팝업에서 전체 녹화 버튼과 전체 스크린샷 버튼 표시 여부를 각각 설정할 수 있도록 개선했습니다.

### v0.1.1

- 치지직에서 검색이나 화면 전환으로 플레이어 크기가 바뀌어도 선택한 영역 비율을 유지하고 녹화가 잘못 종료되지 않도록 개선했습니다.
- 다른 채널 이동, 방송 종료처럼 녹화 중인 스트림이 끝나는 경우 자동으로 녹화를 마치도록 개선했습니다.
- 선택 영역 스크린샷 캡처와 PNG 저장 기능을 추가했습니다.
